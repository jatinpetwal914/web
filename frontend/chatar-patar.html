How to launch this web app locally

  1. Install dependencies: npm install in project root.
  2. Start the backend server: node backend/server.js (or npm start if configured).
  3. Open the frontend: serve the frontend/ folder (e.g. npx serve frontend) or open frontend/index.html in a browser.
  4. By default API runs on http://localhost:5000; if you run the frontend through a static server, the pages will call that API.
